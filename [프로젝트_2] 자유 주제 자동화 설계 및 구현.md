# \[프로젝트 2\] 자유 주제 자동화 설계 및 구현

## 1. 자동화할 반복 업무 정의

## **\[AI 기반 강의 신청 자동관리 시스템\]**

강의 신청이 접수될 때마다 신청 내용을 확인하고, 신청자의 정보를 검토한
후 승인 여부를 판단하여 안내 메일을 발송하는 업무는 반복적으로 수행되는
작업이다. 기존에는 관리자가 신청 내용을 하나씩 확인하고 승인 여부를
판단한 뒤 메일을 작성하고 발송하며, 처리 결과를 관리 시트에 직접
기록해야 했다.

이를 자동화하기 위해 Google Form으로 접수된 신청 정보를 Google Sheets와
연동하고, OpenAI가 신청 내용을 분석하여 승인 여부를 판단하도록
구성하였다. 이후 결과에 따라 신청자와 관리자에게 자동으로 메일을
발송하고, 관리 시트의 처리 상태까지 자동으로 업데이트하는 워크플로우를
구현했다.

------------------------------------------------------------------------

## 2. 자동화 도구 선정 및 선정 이유

### 선정 도구 : n8n

n8n은 다양한 서비스와의 연동이 가능하며, 복잡한 자동화 워크플로우를
시각적으로 설계할 수 있는 오픈소스 자동화 플랫폼이다.

이번 프로젝트에서는 Google Forms, Google Sheets, OpenAI, Gmail을 하나의
워크플로우로 연결해야 했기 때문에 다양한 노드를 자유롭게 구성할 수 있는
n8n을 선정하였다. 또한 IF 노드를 활용한 조건 분기와 Google 서비스 연동이
쉬워 강의 신청 자동 관리 시스템을 구현하기에 적합하다고 판단했다.

------------------------------------------------------------------------

## 3. 기능 요구사항

| 구분         | 기능                                                           |
| ---------- | ------------------------------------------------------------ |
| Trigger    | Google Form 신청이 접수되면 Google Sheets에 새로운 행이 추가되고 워크플로우가 실행된다. |
| 신청 내용 분석   | OpenAI가 신청 목적을 분석하여 승인 또는 보완 필요로 분류한다.                       |
| 조건 분기      | IF 노드를 이용하여 승인(True)과 보완 필요(False)로 자동 분기한다.                 |
| 신청자 메일 발송  | 분류 결과에 따라 승인 안내 또는 보완 요청 메일을 자동 발송한다.                        |
| 관리 시트 업데이트 | Google Sheets의 처리상태를 승인 또는 보완 필요로 자동 변경한다.                   |
| 관리자 알림     | 신청 처리 결과를 관리자 이메일로 자동 발송하여 신청 현황을 확인할 수 있도록 한다.              |
